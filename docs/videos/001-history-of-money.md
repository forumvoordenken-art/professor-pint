Within a generation, nearly every Greek city from southern France to the Black Sea was minting their own coins."

## Asset Productie Workflow (GPT Scene-First)

> Voor elke scene genereer je eerst een complete scene-illustratie via ChatGPT, dan extraheer je individuele elementen.

### Stap 1: Scene-referentie genereren

Geef ChatGPT de volgende prompt per scene:

```
Create a wide illustration in Kurzgesagt style for an educational video scene.

Scene: [beschrijving uit dit document]
Mood: [warm/dramatic/mysterious/etc.]
Color palette: [specifieke kleuren]
Elements that must be visible: [lijst van assets in de scene]

Style rules:
- Flat colors, maximum 16 colors total
- No gradients, no textures, no photorealism
- Bold outlines, geometric simplification
- Characters have simple dot eyes and minimal features
- Landscape orientation, 1920x1080 composition
- Oil painting / storybook feel
```

### Stap 2: Nieuwe elementen extraheren

Voor elk NIEUW element in de scene:

```
Now create ONLY the [element naam] from the previous scene, isolated on a pure white background.

Rules:
- Exact same style, colors, and proportions as in the scene
- Pure white (#FFFFFF) background, no shadows, no ground
- Element centered in frame with padding
- Same flat-color Kurzgesagt style
- PNG format, transparent-ready composition
```

### Stap 3: Vectoriseren

Upload elk geïsoleerd element naar vectorizer.ai:
- Output: SVG
- Detail level: medium (doel: 300-500 paden)
- Controleer dat kleuren overeenkomen met de scene-referentie

### Stap 4: Animeren in Remotion

Claude animeert het SVG-element in Remotion als React component met:
- `withAssetPaint` HOC voor oil painting effect
- Idle animaties (subtle movement)
- Talking animaties (voor characters)
- Emotion states (voor Professor Pint)
